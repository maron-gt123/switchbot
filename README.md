## switch bot APIを活用した湿温度計情報の定期監視
*  switch botのAPIを活用した電力/湿度/温度をGrafanaを用いて監視を実施する
  * 構築要件
    *  switch bot API 1.1を使用
    *  switch botのデバイスは自己解決
    *  本内容は監視を目的としており制御に関してはサポート外とする
  *  調達デバイスのIDを取得
